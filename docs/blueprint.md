# **App Name**: JEE Prep Ace

## Core Features:

- Content Delivery: Deliver pre-recorded video lectures and study materials categorized by topic and difficulty level.
- Live Classes: Schedule and conduct live interactive classes with integrated chat and screen sharing.
- Practice Questions: Provide a bank of practice questions with varying difficulty levels and detailed solutions.
- Mock Tests: Simulate the IIT JEE exam environment with timed mock tests and detailed performance analysis.
- Doubt Resolution: Enable students to post doubts and receive timely solutions from instructors.
- Personalized Learning Paths: AI-powered tool analyzes student performance to suggest personalized learning paths and focus areas. This recommendation engine looks at the student's test performance to guide students on where they can improve.
- Progress Tracking: Track student progress, scores, and time spent on each module.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082), representing knowledge and focus.
- Background color: Light lavender (#E6E6FA), for a calming and clean interface.
- Accent color: Electric purple (#BF00FF) to highlight key interactive elements.
- Body text: 'Inter', a grotesque-style sans-serif with a modern look suitable for body text.
- Headline text: 'Space Grotesk', a sans-serif with a computerized, techy feel, used to make the page headings easily readable.
- Use clear, minimalist icons for navigation and to represent different subjects/modules.
- Clean and structured layout for easy navigation, using cards to separate the topics. Maximize space so as not to overwhelm users.
- Subtle animations to highlight important actions, such as completing a module or submitting a quiz.
